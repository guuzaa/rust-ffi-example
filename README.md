# C & Rust FFI Example

This is a simple example of how to use C and Rust together using FFI.

## Introduction

1. Compile C code into a static library.
2. Generate Rust bindings using `bindgen`.
3. Build the Rust code.

## Prerequisites

- Cargo
- CMake 3.15 or higher

## Building

```bash
cargo build
```

## Testing

```bash
cargo test
```
